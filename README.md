# 3D Sound & Entertainment

This is the official website for **3D Sound & Entertainment**, built from the ground up as a passion project to showcase everything we do as a DJ and event entertainment company. The site serves as the digital home for the entire 3D Sound roster, featuring our music, our talent, booking information, and everything clients need to know about bringing the energy to their next event.

This was not built from a template. Every line of code was designed and developed by our own team to reflect who we are and how we move.

## Features

- **Fully responsive design** that looks and feels great on desktop, tablet, and mobile, with five layout breakpoints and reduced motion support for accessibility
- **Custom audio player** built from scratch using HTML5 audio, featuring play/pause, previous/next, a seekable progress bar, time display, a Now Playing overlay with the full track list, and animated equalizer bars
- **VIB3S album integration** with all 26 mastered tracks playable directly on the site, complete with album artwork and track duration loading
- **HoneyBook booking form** embedded on a dedicated bookings page, allowing clients to submit event inquiries directly
- **Artist inquiry tracking** that carries the selected artist name from the talent section into the booking flow
- **Talent roster display** showcasing 15 DJs and 3 MCs with hover overlays, Instagram links, and direct booking buttons
- **Testimonials carousel** that auto rotates every five seconds with dot navigation, touch swipe support, and hover pause on desktop
- **Smooth scroll navigation** with a sticky header, fullscreen overlay menu, and active section highlighting as you scroll
- **Performance optimized** with WebP images, lazy loading, font preloading, and zero external JavaScript dependencies
- **SEO ready** with Open Graph tags, Twitter Card meta, canonical URLs, and semantic HTML throughout

## Tech Stack

| Layer       | Technology                                                                |
|-------------|---------------------------------------------------------------------------|
| Markup      | HTML5 with semantic elements                                              |
| Styling     | Custom CSS3 with CSS variables, flexbox, grid, and clamp() for fluid type |
| Scripting   | Vanilla JavaScript (no frameworks or libraries)                           |
| Fonts       | via Google Fonts (weights 200 through 800)                                |
| Booking     | HoneyBook embedded form                                                   |
| Hosting     | GitHub Pages with custom domain                                           |
| Images      | WebP format with lazy loading and priority hints                          |

No build tools, no bundlers, no npm packages. Just clean HTML, CSS, and JavaScript.
